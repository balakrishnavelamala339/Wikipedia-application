# Wikipedia-application

Wikipedia Search Application

Project Overview
The Wikipedia Search Application is a simple web-based app that allows users to search for articles from Wikipedia. It fetches real-time data using the Wikipedia API and displays search results with titles, links, and descriptions.
🚀 Features
🔍 Search Wikipedia articles in real-time
📄 Display article title, snippet, and link
🌐 Open full article in a new tab
⚡ Fast and responsive UI
📱 Mobile-friendly design
🛠️ Technologies Used
HTML
CSS
JavaScript
Wikipedia API
📂 Project Structure

wikipedia-app/

⚙️ How It Works
User enters a keyword in the search box
App sends a request to Wikipedia API
Receives JSON data
Displays results dynamically on the page
🔗 API Used
Wikipedia Search API
Endpoint:

https://en.wikipedia.org/w/api.php?action=query&list=search&srsearch=KEYWORD&format=json&origin=*
